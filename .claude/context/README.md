# Context Directory

This directory contains project context documentation for AI agents.

## Purpose

- **Fast Agent Onboarding**: Quick understanding of project scope and patterns
- **Project Continuity**: Maintain context across development sessions
- **Consistent Understanding**: Same reference for all contributors
- **Living Documentation**: Updated as the project evolves

## Files

### Foundation
- `project-brief.md` - High-level project description and goals
- `project-overview.md` - Detailed project scope and features
- `progress.md` - Current state and recent changes

### Technical
- `tech-context.md` - Technology stack and dependencies
- `project-structure.md` - Directory organization
- `system-patterns.md` - Architecture and design patterns

### Product
- `product-context.md` - User needs and product requirements

## Usage

### Starting a Session
Read the CLAUDE.md files first, then relevant context files based on your task.

### After Significant Changes
Update `progress.md` with what was accomplished.

### When Adding Features
Reference `system-patterns.md` to follow established patterns.
