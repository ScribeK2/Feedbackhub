# FeedbackHub Project Brief

## Overview

FeedbackHub is a feedback collection and management application built with Rails 8 and Phlex.

## Goals

- Collect user feedback efficiently
- Organize and categorize feedback
- Enable team collaboration on feedback items
- Provide insights from collected feedback

## Technology Choices

### Why Rails 8?
- Mature, battle-tested framework
- Excellent developer productivity
- Built-in conventions reduce decisions

### Why Phlex?
- Type-safe, Ruby-native views
- Better performance than ERB
- Composable components
- First-class IDE support

### Why PhlexyUI/DaisyUI?
- Pre-built, accessible components
- Tailwind CSS integration
- Consistent design system
- Rapid UI development

### Why SQLite + Solid Suite?
- Zero-config database
- Production-ready with Solid Queue/Cache/Cable
- Simplified deployment
- Perfect for single-server apps

## Principles

1. **Simplicity First**: Choose the simplest solution that works
2. **Convention Over Configuration**: Follow Rails conventions
3. **Progressive Enhancement**: Work without JS, enhance with Turbo/Stimulus
4. **Test Everything**: Comprehensive test coverage
